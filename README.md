# Internship-Task8
Objective
Built an interactive dashboard to analyze Sales and Profit by Region, Category, Sub-Category, and Time.

Tool Used
Power BI
Dataset
Superstore Sales Dataset (CSV)

Dashboard Includes
Line Chart → Sales over time (Year/Month)
Donut Chart → Profit by Region
Bar Chart → Sales by Category & Sub-Category
Slicer → Sub-Category filter
KPI Cards:
Total Sales: 211.35K
Total Profit: 63.48K

Key Insights
Total Sales reached 211.35K with 63.48K Profit.
East and West regions contribute the highest profit share.
Technology and Office Supplies categories generate strong sales.
Sub-category filtering shows Chairs and Accessories perform well.

What This Dashboard Shows
Sales trend over time
Regional profit comparison
Category-level performance
Interactive filtering for deeper analysis
